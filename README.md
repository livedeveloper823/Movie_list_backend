# APIs
## 1. Videos
### Get all video data

- GET 
```http://host:5000/api/videos/all```

### Get video data with video_id
- GET
```http://host:5000/api/videos/video/:video_id```

### Update video data with video_id

- PUT 
```http://host:5000/api/videos/video/:video_id```


### Add video

- POST
```http://host:5000/api/videos/addvideo```

## 2. Auth

- POST
```http://host:5000/api/auth/login```
